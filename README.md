# AI Resume Analyzer 🤖

An AI-powered resume analyzer built with FastAPI and React.

## Features
- AI-powered CV analysis
- Score system
- Job description matching
- AI chat
- PDF report download
- User login/signup

## Tech Stack
- Backend: FastAPI, Python
- Frontend: React
- AI: Groq/Mistral API
